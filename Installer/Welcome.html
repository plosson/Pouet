VirtualMic Audio Driver

This installer will add a VirtualMic virtual microphone to your Mac.

After installation:

  • Open System Settings → Sound → Input — you'll see VirtualMic listed.
  • Select it as your microphone in any app (Zoom, Discord, FaceTime …).
  • Use the VirtualMicApp inject audio.mp3 command to play audio through it.

The driver runs entirely in user space. No kernel extension is installed. No System Integrity Protection changes are required.
