Quick Start

# Inject an MP3 (plays once then stops)
VirtualMicCli inject /path/to/music.mp3

# Loop an MP3 continuously (Ctrl-C to stop)
VirtualMicCli stream /path/to/music.mp3

# Check buffer fill
VirtualMicCli status

# Clear injected audio
VirtualMicCli stop

Select VirtualMic as your microphone input in any application.
